Mailbox Menu

Create

 Create a new mailbox on the server. The server window must be the top window.

Rename

 Change the name of the selected mailboxes.

Mark Contents Seen

 Marks any unseen messages as seen.

Delete Mailbox

 Delete the selected mailboxes on the server.

Expunge

 Permanently remove deleted messages.

Synchronise

 Synchronises messages between server-based mailboxes and their local disconnected counter-parts.

Clear Disconnected

 Clears any messages stored locally in a disconnected mailbox.

Display Hierarchy

 Commands related to display hierarchies.

Cabinets

 Commands related to cabinets in the main server window.

Subscribe

 Subscribe to the selected mailboxes.

Unsubscribe

 Unsubscribe from the selected mailboxes.

Check Email

 Look for new messages in selected mailboxes.

Check Favourites

 Look for new messages in mailboxes marked for automatic checking.

Sort By

 Sort the messages in the top mailbox window.

Goto Message

 Selects a specific message in the active mailbox window.

Load All Messages

 Loads all messages into the mailbox cache of the active mailbox window.

Update Current Match

 Updates the current set of matched messages based on the current Match criteria in effect.

Search Mailbox

 Search for messages in mailboxes.

Search Again

 Search again for messages in selected mailboxes.

Next Search

 Open the next mailbox which matched the last search.

Rules

 Display the Rules manager window for use with creating and editing rules and filters.

Apple Rules

 Apply the selected rule to the mailbox or selected messages.

Make Rule

 Create a rule based on characteristics of the selected messages.
