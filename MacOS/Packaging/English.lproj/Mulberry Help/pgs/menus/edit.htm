Edit Menu

Undo

 Undo the last action.

Redo

 Redo the last action

Cut

 Cut the selection and put it on the clipboard.

Copy

 Copy the selection and put it on the clipboard.

Paste

 Insert clipboard contents.

Delete

 Delete the selection.

Select All

 Select all items or text.

Find

 Find text.

Find Next

Find Selection

Replace

Replace & Find Next

Replace All

 Replace all matching text.

Text Macros

 Edit text macros.

Details

 Change the properties of the currently selected items.

Change Password

 Allows passwords for accounts to be changed.

Speak Message
