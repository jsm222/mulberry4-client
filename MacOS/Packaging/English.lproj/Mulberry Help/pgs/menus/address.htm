Address Menu

Address Book Manager

 Brings the Address Book Manager to the front.

New Address Book

 Create a new address book. A local or remote server must be selected in the Address Book Manager window.

Open Address Book

 Open an address book.

Rename Address Book

 Renames an address book.

Import Addresses

 Imports addresses in a different format address book file.

Export Addresses

 Export address from an address book into another format.

Delete Address Book

 Deletes an address book.

Login

 Logs in to the selected address book server.

Logout

 Logs out of the selected address book server.

Refresh List

 Refreshes the list of address books for the selected server.

Synchronise

 Synchronises the selected address books in the address book manager window.

Clear Disconnected

 Removes the disconnected cache files for the selected address books in the address book manager window.

Expand Address

 Expands an address in an address entry field by doing a search of address books.

Search for Addresses

 Searches for addresses.
