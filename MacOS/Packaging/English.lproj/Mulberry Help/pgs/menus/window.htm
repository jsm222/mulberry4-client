Window Menu

Save Default Window

 Save the state of the current window as the new default.

Reset to Default

 Reset the state of the current window to its default.

Collapse/Expand Window Header

 Collapses or expands the active window's header area.

Show/Hide Parts

 Collapses or expands the active window's parts list.

Show/Hide Quick Search

 Shows or hides the quick search area in a mailbox pane.

Toolbars

 Manages the toolbars in all windows.

Layout

 Changes the 3-pane/separate window layouts.

Hide/Show List

 Shows or hides the mail or address account pane in the 3-pane window.

Hide/Show Items

 Shows or hides the mailbox or address book pane in the 3-pane window.

Hide/Show Preview

 Shows or hides the message or address/group preview pane.

Options

 Allows options related to the front most window to be changed.

Status

 Show or hide the status window.

Window List

 List of all open windows.
