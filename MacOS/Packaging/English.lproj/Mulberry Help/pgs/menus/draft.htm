Draft Menu

Include Original Message

 Insert the text of the original message.

Insert Signature

 Inserts the signature at the current selection point in the current draft.

Add Cc Address Field

 Add the Cc address field to the draft window if not already present.

Add Bcc Address Field

 Add the Bcc address field to the draft window if not already present.

Attach File

 Attach a file to the current draft.

New Text Part

 Add a new text part to the current draft.

New Multipart

 Add a new multipart to the current draft.

Current Text Part

 Change the format of the current text part.

Send

 Send the current draft message.

Append to Mailbox

 Copy the current draft message to a mailbox.

Append Now

 Copy the current draft message to the selected 'Copy To' mailbox .

Also Copy Original Message

 When turned on, the original message of a reply will be copied into the chosen Copy to mailbox immediately before the copy of the reply being sent.

Sign

 Turns on verification of the message using a secure signature (only if a suitable plugin is available).

Encrypt

 Turns on encryption of the message (only if a suitable plugin is available).

Notifications

 Send the current draft message.

Wrap Lines

 Inserts carriage returns at the end of each selected line.

Unwrap Lines

 Removes carriage returns at the end of each selected line.

Quote Lines

 Inserts the quotation string at the start of each selected line.

Unquote Lines

 Removes the first quotation string at the start of each selected line.

Requote Lines

 Rewraps and requotes the selected lines.

Shift Left

 Removes a tab, or spaces, from the start of each selected line.

Shift Right

 Inserts a tab, or spaces, at the start of each selected line.

Font

 Changes the font of the selected text in the current draft.

Size

 Changes the font size of the selected text in the current draft.

Style

 Changes the font style of the selected text in the current draft.

Alignment

 Changes the alignment of the selected text in the current draft.

Colour

 Changes the colour of the selected text in the current draft.
