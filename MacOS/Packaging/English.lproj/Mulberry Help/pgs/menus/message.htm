Message Menu

Read Previous

 Display the previous message.

Read Next

 Display the next message.

Delete and Read Next

 Delete the current message and display the next one.

Copy and Read Next

 Copy the current message and display the next one.

Reply

 Reply to the current message

Reply to Sender

 Reply to the 'Sender' address of the current message.

Reply to From

 Reply to the 'From' address of the current message.

Reply to All

 Reply to all address in the current message.

Forward

 Forward the current message.

Redirect

 Bounce the current message.

Reject

 Reject the current message to make it look like it was not delivered.

Send Again

 Send the current message again.

Create Digest

 Create a message digest out of the selected messages.

Move/Copy to Mailbox

 Copy the current message to another mailbox.

Move/Copy Now

 Copy the current message to the selected 'Copy To' mailbox.

Flags

 Change the flags for the current message.

Extract Parts

 Extract the selected parts in the current message to disk.

Decode Embedded

 Decode embedded Binhex or UU parts in the current message.

Verify/Decrypt

 Verifies or decrypts a signed or encrypted message (only if a suitable plugin is available).

Delete

 Delete the current message.

Show Header

 Toggle the header display of the current message's header.
