How do I make the message and draft windows remember their size and position?

  1. Setup the message or draft window to the size and position that you want as the default.
  2. Make sure you have the header and parts list collapsed or expanded for the default state you want.
  3. Select the 'Save Default Window' command from the Windows menu. This will save the current (top) window state as the default for that type of window
