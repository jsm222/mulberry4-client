Message Window Toolbar

 The Draft button will create a new empty draft message.
The Reply button allows you to create drafts for replying to the selected messages in the mailbox pane or the current message in the preview pane. If there is more than one selected message, you will be given the choice of replying to each one individually, or replying to them all in one draft. Use of the option key will toggle the display of the Reply addressing dialog that allows choice of which addresses to reply to when creating the reply draft.
Similar to the Reply button except that the reply is sent to all correspondents in the original message.
The Forward button allows you to create drafts for forwarding the selected messages in the mailbox pane or the current message in the preview pane. If there is more than one selected message, you will be given the choice of forwarding each one individually, or forwarding them all in one draft. Use of the option key will toggle the display of the Forwarding dialog that allows choice of various options related to creating the forward draft.
The Redirect button allows you to redirect a message to someone else.
Copy or Move the selected messages in the mailbox pane or the current message in the preview pane to another mailbox. Click on the right-side to bring up a popup menu to choose the mailbox to copy/move to - the copy/move is done when you select the mailbox. Click on the left-side to copy/move to the previously chosen mailbox.
The Delete button allows you to delete messages in the mailbox pane or the message in the preview pane. A deleted message can be permanently removed from the mailbox by clicking the Expunge button. If the selected message is already deleted, this button will be labelled Undelete and it will undelete the message when clicked.
The Print button prints the current text.
The Del-Nxt deletes the current message and switches the display to the next message in the mailbox.
The Prev button switches the display to the previous message in the mailbox.
The Next button switches the display to the new message in the mailbox.
