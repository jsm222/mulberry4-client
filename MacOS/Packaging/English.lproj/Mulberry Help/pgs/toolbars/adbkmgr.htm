Address Book Manager Window Toolbar

 The Draft button will create a new empty draft message.
The New button will create a new address book.
The Open button will open the selected address books.
The Delete button will delete the selected address books.
The Search button will display the Address Search window to allow addresses to be searched for in selected address books or directory servers.
The Details button is used to show a dialog containing useful information and options that control the behaviour of any selected address books.
