Server Window Toolbar

 The Draft button will create a new empty draft message.
The Login/Logout button allow you to login or logout off the selected accounts in the server pane.
The Create button allows you to create new mailboxes in the server pane.
The Cabinet button controls the display of cabinets in the server pane. It also allows user-defined cabinets to be created or removed.
The Check button causes the mailbox in the mailbox pane to be immediately checked for new mail, or if the server pane is focussed, it will initiate and immediate check of all mailboxes marked for checking.
The Search button displays the search window allowing you to search for messages matching selected criteria in the mailbox.
The Details button is used to show a dialog containing useful information and options that control the behaviour of the mailbox.
