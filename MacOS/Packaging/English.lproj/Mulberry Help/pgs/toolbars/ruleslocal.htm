Rules Window Toolbar - Local tab

Rule
 The Rule button will create a new rule.
Trigger
 The Trigger button will create a new trigger.
Edit
 The Edit button edits the selected rule or trigger.
Delete
 The Delete button deletes the selected rule or trigger.
Apply
 The Apply button applies any rules that are available in the Apply Rules menu to all open mailboxes.
Show Triggers
 The Show/Hide Triggers buttons shows or hides the triggers list area.
