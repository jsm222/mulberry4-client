Draft Window Toolbar

 The Draft button will create a new empty draft message.
The Save button allows the current draft to be saved to disk or to a mailbox, depending on the state of the 'Save Draft' preferences in the Outgoing preferences panel.
The Attach button allows you to choose files to attach to the current draft.
The Spelling button starts spell checking of the text in the current draft..
Adds a Cc: address field for including additional recipients.
Adds a Bcc: address field for blind-copying additional recipients.
The Sign button turns on or off the addition of a cryptographic digital signature to the draft. This can be used by recipients to verify the authenticity of the message. This option will only be present if cryptographic support is enabled by the addition of suitable plugins.
The Encrypt button turns on or off the addition of a cryptographic encryption of the draft. This can be used to encrypt the content of the message so that only the specified recipients of the message can decrypt and read the content. This option will only be present if cryptographic support is enabled by the addition of suitable plugins.
The Send button will send the message. If you have outgoing queues enabled, the draft will be put into a queue for sending in the background. If queues are disabled, the draft will be sent immediately possibly resulting in a delay as it is sent.
