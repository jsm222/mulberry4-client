Shortcuts

  • The Server (mailbox account) pane
  • The Mailbox pane
  • The Message Preview pane
  • The Message Window
  • The Draft Window
  • The Address Book pane

Server Pane
To move the selection to the next mailbox containing new message Type tab on the keyboard
To expand a selected row Type right-arrow on the keyboard
To collapse an expanded selected row Type left-arrow on the keyboard
To move to the hierarchical parent of a collapsed selected row Type left-arrow on the keyboard
To logoff the server corresponding to the selected row Type delete on the keyboard
To remove the selected Cabinet item Type delete on the keyboard
To remove the selected Cabinet item content Type delete on the keyboard
To select an item in the list Start typing its name

Mailbox Pane
To move the selection to the next new message or move to the next mailbox containing new messages Type tab on the keyboard
To move the selection to the previous new message Type shift-tab on the keyboard
To move to the next mailbox containing new messages Type option-tab on the keyboard
To delete or undelete the selected messages Type delete on the keyboard
To scroll the preview pane to the bottom (if one is visible) and then move the selection to the next message Type space on the keyboard
To automatically Match a message based using some criteria of that message Option-click on the table cell that contains the data to match on for that message

When using Thread sorting:
To move the selection to the next message at the same level in the thread hierarchy Type right-arrow on the keyboard
To move the selection to the previous message at the same level in the thread hierarchy, or to the parent if no previous message exists Type left-arrow on the keyboard

Message Preview Pane
To scroll the text to the bottom and then move to the next message Type space on the keyboard
To scroll the text to the bottom, delete the current message, and then move to the next message Type option-space on the keyboard
To move to the next new message Type tab on the keyboard
To delete the current message and then move to the next message Type delete on the keyboard

Message Window
To scroll the text to the bottom and then move to the next message Type space on the keyboard
To scroll the text to the bottom, delete the current message, and then move to the next message Type option-space on the keyboard
To delete the current message and then move to the next message Type delete on the keyboard

Draft Window
To expand a nick-name into an address, or to search for addresses matching some text Type the nick-name or text to search for into an address field and type tab or return on the keyboard

Address Book Window
To select and address using the text for the currently selected sort method Start typing the text
